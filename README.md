# 云音乐复刻 uni-app 最终优化版

本项目适合作为 HBuilderX + uni-app 课程作业，包含：

- 首页：推荐页
- 搜索页：分类发现页
- 笔记页：双列瀑布流
- 我的页：个人中心
- 登录页：学号 + 姓名 + 自定义密码登录
- 公共组件：底部导航、底部迷你播放器

## 本次优化内容

1. H5 电脑浏览器中自动限制为手机 App 宽度，避免页面横向铺满导致效果变形。
2. 隐藏横向滚动条，页面更接近手机 App 展示效果。
3. “我的”页面改成通用账号信息，不再使用个人姓名、个人学号或个人昵称。
4. 登录缓存 key 已更换，避免浏览器中残留旧账号信息。
5. 保留老师要求的登录字段：学号、姓名、密码。

## 默认测试账号

默认测试账号在 `common/data.js` 中：

```js
export const STUDENT = {
  id: '2024000000',
  name: '学生姓名',
  password: '123456'
}
```

提交前可以把这三个值改成自己的：

- id：你的学号
- name：你的姓名
- password：自定义密码

## 运行方式

1. 用 HBuilderX 打开项目文件夹。
2. 运行到浏览器或手机模拟器。
3. 点击底部“我的”。
4. 点击“去登录”。
5. 使用 `common/data.js` 里的账号信息登录。

## 素材替换

项目使用 `static/images` 中的占位图。若想更像截图，可以替换这些图片，文件名保持不变即可，无需修改代码。
